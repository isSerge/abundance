Struct OwnedAlignedBuffer

pub struct OwnedAlignedBuffer { /* private fields */ }
Expand description

Owned aligned buffer for executor purposes.

See SharedAlignedBuffer for a version that can be cheaply cloned, while reusing the original allocation.

Data is aligned to 16 bytes (128 bits), which is the largest alignment required by primitive types and by extension any type that implements TrivialType/IoType.
